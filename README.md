# 🚀 HackerRank Challenges

Bem-vindo ao meu repositório de soluções para desafios do HackerRank! Este é um espaço onde documento minha jornada de aprendizado e aprimoramento em programação através da resolução de problemas algorítmicos.

## 📋 Sobre

Este repositório contém minhas soluções para diversos desafios do [HackerRank](https://www.hackerrank.com/), organizadas por categoria e nível de dificuldade. Cada solução inclui uma breve explicação da abordagem utilizada e a complexidade do algoritmo quando relevante.

## 📝 Como Usar Este Repositório

1. Navegue até a categoria de interesse
2. Cada pasta contém:
   - Código da solução
   - Link para o desafio original
   - Explicação da abordagem
   - Complexidade temporal e espacial (quando aplicável)
